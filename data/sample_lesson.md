# サンプル講義：プログラミングの基本

## 1. 変数とは
プログラムでは「変数」という箱を使って値を保存します。
```python
x = 10
print(x)
```

## 2. 条件分岐

条件によって処理を変えるには if 文を使います。

```python
if x > 5:
    print("大きい")
else:
    print("小さい")
```

## 3. まとめ

変数と条件分岐はすべてのプログラムの基礎です。
